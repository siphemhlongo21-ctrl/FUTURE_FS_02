# Mini CRM - Client Lead Management System

## Future Interns Task 2 | Full Stack Web Development

A simple CRM application to manage client leads generated from website contact forms.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Features
- Login & Logout (Admin access)
- Lead listing (name, email, source, status)
- Lead status updates (New / Contacted / Converted)
- Notes and follow-ups for each lead
- Analytics dashboard
- CSV Export

## How to Run
1. cd backend
2. npm install
3. node index.js
4. Open index.html in browser
